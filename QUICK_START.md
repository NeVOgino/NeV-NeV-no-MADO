# クイックスタートガイド

## 📖 閲覧方法

1. `index.html` をダブルクリック
2. タブで「全員向け」「職員向け」を切り替え
3. 検索ボックスで情報を検索

### 🔍 検索機能の使い方

**基本的な使い方:**
1. 画面上部の検索ボックス（🔍 検索...）にキーワードを入力
2. 入力すると該当するセクションのみが表示される
3. 検索ボックスを空にするとすべて表示される

**検索の特徴:**
- リアルタイム検索（入力と同時に結果表示）
- 部分一致（キーワードが含まれていれば表示）
- 大文字小文字を区別しない

**検索例:**
- 「職場環境」→ 職場環境改善に関する項目のみ表示
- 「カレンダー」→ R6/R7年度カレンダーのセクション表示
- 「就業規則」→ 就業規則関連の項目表示（職員向けタブ）

**検索対象:**
- セクション名、日付、タイトル、詳細情報、リンクテキスト

## ✏️ 編集方法（5ステップ）

### ステップ1: 編集画面を開く
- `admin.html` をダブルクリック
- ファイル共有（file://）から開く場合：「📤 data.jsonをアップロード」ボタンでdata.jsonを読み込む
- Webサーバー（http://）から開く場合：自動的にdata.jsonが読み込まれる

### ステップ2: 編集したいセクションの「編集」ボタンをクリック

### ステップ3: 内容を編集
- お知らせを追加する場合は、既存の項目をコピーして追加
- 日付、内容、詳細を変更

### ステップ4: 保存とHTMLの再生成
1. 「保存」ボタンをクリック
2. 「💾 data.jsonをダウンロード」ボタンをクリック
3. ダウンロードしたファイルを元の `data.json` と置き換え
4. **重要**: 以下のコマンドでindex.htmlを再生成
   ```cmd
   cd \\cev-file5\share\NEVの窓ブラウザ版\
   python generate_html.py
   ```

### ステップ5: 確認
- `index.html` を開いて変更を確認

## ⚠️ 注意事項

- **必ずバックアップ**: 編集前に data.json をコピー
- **形式を守る**: カンマ、括弧、クォーテーションに注意
- **段階的に**: 少しずつ編集して保存
