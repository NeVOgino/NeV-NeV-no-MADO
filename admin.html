🪟 NEVの窓 - 編集モード

掲示板の内容を編集できます

📝 編集方法

  1. 編集したいセクションの「編集」ボタンをクリックします
  2. テキストエリアで内容を編集します（JSONフォーマットで）
  3. 「保存」ボタンをクリックして変更を保存します
  4. 「💾 data.jsonをダウンロード」ボタンで編集したデータをダウンロード
  5. ダウンロードしたdata.jsonファイルで元のdata.jsonを置き換えます
  6. 重要： 以下のコマンドを実行してindex.htmlを更新します：
    python generate_html.py
    これにより、data.jsonの内容がindex.htmlに反映されます。
