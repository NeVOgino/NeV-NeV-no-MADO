🪟 NEVの窓 - 編集モード

掲示板の内容を編集できます

📝 編集方法

  1. data.jsonよみこみ： 最初に現在のdata.jsonファイルを読み込みます
  2. 編集： 編集したいセクションの「編集」ボタンをクリックします
  3. 内容変更： テキストエリアで内容を編集します（JSONフォーマット）
  4. 保存： 「保存」ボタンをクリックして変更を保存します
  5. data.jsonかきこみ： 編集したデータをダウンロードして元のdata.jsonを置き換えます
  6. 反映： コマンド python generate_html.py を実行してindex.htmlに反映します

🎯 項目管理機能

各項目にマウスをホバーすると、以下のボタンが表示されます：

  • ↑（上へ移動）： 項目を1つ上に移動します
  • ↓（下へ移動）： 項目を1つ下に移動します
  • 移動： 項目を別のセクションに移動します
  • 削除： 項目を削除します（確認ダイアログが表示されます）

各セクションの下部にある「➕ 項目を追加」ボタンで新しい項目を追加できます。

📁 セクション管理機能

セクション全体を管理するための機能：

  • ➕ セクションを追加： ページ上部のボタンで新しいセクションを追加できます
  • ↑/↓（セクションの並び替え）： セクション名の右側のボタンでセクションの順序を変更できます
  • 🗑️（セクション削除）： セクションとその中のすべての項目を削除します

💡 自動バックアップ： 「かきこみ」ボタンをクリックすると、data.jsonファイルと一緒に、タイムスタンプ付きのバックアップファイル（例：data_backup_20251111_120530.json）が自動的にダウンロードされます。万が一の時に安心です。
