# NEVの窓 - ブラウザ版

エクセルファイルのNEVの窓をブラウザ版に変更しました。

## 📋 概要

NEVの窓は、社内掲示板システムのブラウザ版です。Excelファイルで管理していた情報を、見やすく使いやすいWebインターフェースで閲覧・編集できるようになりました。

## 🚀 使い方

### 閲覧モード

1. `index.html` をブラウザで開きます
2. 「全員向け」と「職員向け」のタブで情報を切り替えられます
3. 検索ボックスで情報を素早く見つけられます

### 編集モード

1. 閲覧画面の「📝 編集モード」ボタンをクリックします
2. 編集したいセクションの「✏️ 編集」ボタンをクリックします
3. 内容を編集して「💾 保存」をクリックします
4. **重要**: 「💾 data.jsonをダウンロード」ボタンでファイルを保存します
5. ダウンロードしたファイルを元の `data.json` と置き換えます

## 📁 ファイル構成

- `index.html` - メインの閲覧画面
- `admin.html` - 編集画面
- `style.css` - デザイン（共通）
- `admin.css` - 編集画面のデザイン
- `script.js` - 閲覧画面の動作
- `admin.js` - 編集画面の動作
- `data.json` - 掲示板のデータ（これを編集します）

## 🔧 編集の仕方（詳細）

### 簡単な編集方法

1. `admin.html` を開く
2. 編集したいセクションの「編集」ボタンをクリック
3. 表示されたテキストエリアで内容を編集
   - 日付、内容、詳細などを変更できます
   - 項目を追加する場合は、同じ形式でコピーして追加します
4. 「保存」をクリック
5. 「data.jsonをダウンロード」でファイルを保存
6. ダウンロードしたファイルを元のdata.jsonと置き換え

### data.jsonの構造

```json
{
  "全員向け": {
    "title": "全員向け情報",
    "sections": [
      {
        "name": "INFORMATION",
        "items": [
          {
            "date": "2025.9.27",
            "content": "お知らせのタイトル",
            "detail": "詳細情報"
          }
        ]
      }
    ]
  }
}
```

## 💡 コーディング初心者の方へ

- **編集画面を使う**: 直接ファイルを編集せず、編集画面（admin.html）を使うことをお勧めします
- **バックアップ**: 編集前に必ずdata.jsonのバックアップを取ってください
- **形式を守る**: JSON形式を壊さないよう、既存の項目をコピーして修正してください
- **エラーが出たら**: 「JSONの形式が正しくありません」と表示された場合は、カンマ(,)や括弧が正しいか確認してください

## 🌐 ウェブサーバーでの公開

ローカルで開くだけでなく、Webサーバーに配置することも可能です：

1. すべてのファイルをWebサーバーにアップロード
2. `index.html` にアクセスして閲覧
3. 編集は `admin.html` にアクセス
