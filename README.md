# NeVの窓 - 職員向け情報ポータル

エクセルファイルのNeVの窓をブラウザ版に変更したウェブサイトです。

## 🌐 ウェブサイトについて

このリポジトリは、NeV職員向けの情報ポータル「NeVの窓」のウェブ版です。以下の情報を提供しています：

- **全員向け**: お知らせ、共通コーナー、会議室予約、各部掲示板
- **職員向け**: 標準書、各種帳票フォーマット、職員向け情報

## 📝 更新方法

コーディング初心者の方でも簡単に更新できます！

### マニュアル

- **[更新マニュアル.md](./更新マニュアル.md)** - ウェブサイトの内容を更新する方法
- **[資料ファイルの管理ガイド.md](./資料ファイルの管理ガイド.md)** - PDF、Excel等の資料ファイルを保存してリンクする方法
- **[GitHub-Pages-セットアップガイド.md](./GitHub-Pages-セットアップガイド.md)** - GitHub Pagesの設定方法（管理者向け）

### 簡単な更新手順

1. GitHubにログイン
2. `index.html` を開く
3. 鉛筆マーク（✏️）をクリックして編集
4. 変更を保存（Commit）

## 🚀 GitHub Pages での公開

このサイトはGitHub Pagesで自動公開されています。

**公開URL**: `https://NeVOgino.github.io/NeV-NeV-no-MADO/`

変更を保存すると、数分で自動的にウェブサイトに反映されます。
