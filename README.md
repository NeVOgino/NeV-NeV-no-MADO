# NeV-NeV-no-MADO
エクセルファイルのNeVの窓をブラウザ版に変更する

## バックアップデータの管理

### バックアップデータの保存先
バックアップデータを生成したら、必ず`json_backup`フォルダに保存してください。

### バックアップファイルの命名規則
バックアップファイル名には、作業内容を簡潔に反映させてください。

**命名例：**
- `json_backup/2025-11-12_初期設定.json`
- `json_backup/2025-11-12_レイアウト変更.json`
- `json_backup/2025-11-12_データ追加.json`

**推奨フォーマット：**
```
json_backup/YYYY-MM-DD_作業内容の簡潔な説明.json
```

これにより、後からバックアップファイルを探す際に、どの時点でどのような作業を行ったかを容易に把握できます。
